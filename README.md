AI 기반 후쿠시마 여행일정 추천사이트 (SANSANTRIP)
### ⚙️ Setup
```
1. 프로젝트 clone
$ git clone 
3. 프로젝트 폴더로 이동
$ cd 
4. 패키지 설치
$ npm install
5. 앱 실행
$ npm start
```
